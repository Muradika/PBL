SIPAk (Sistem Informasi Pengumuman Akademik Online) adalah aplikasi berbasis web yang digunakan untuk mengelola dan menyebarkan pengumuman akademik kampus secara terpusat, cepat, dan terorganisir.

Aplikasi ini dirancang untuk memudahkan **mahasiswa**, **dosen**, dan **admin** dalam mengakses serta mengelola informasi akademik seperti pengumuman, jadwal, dan informasi kemahasiswaan.
